Política de Privacidade

Última atualização: 27/02/2026

1. Informações Coletadas

Coletamos informações fornecidas pelo usuário, como nome, e-mail e dados necessários para o funcionamento do aplicativo.

2. Uso das Informações

As informações são utilizadas para autenticação, processamento de pedidos e melhoria dos serviços.

3. Compartilhamento

Não compartilhamos dados pessoais com terceiros, exceto quando necessário para cumprimento legal.

4. Armazenamento e Segurança

Os dados são armazenados em ambiente seguro, utilizando criptografia e controle de acesso.

5. Público Infantil

Este aplicativo não é destinado a menores de 13 anos e não coletamos intencionalmente dados de crianças.

6. Contato

Para dúvidas, entre em contato pelo e-mail: seuemail@empresa.com
